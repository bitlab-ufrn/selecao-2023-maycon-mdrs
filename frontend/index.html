Conteúdo Impróprio DETECTOR

INSTRUÇÕES

O site "Conteúdo Impróprio DETECTOR" tem como objetivo verificar textos em busca de palavras inadequadas, incluindo palavrões e termos ofensivos. A interface do site apresenta uma postagem semelhante à de uma rede social, permitindo que o usuário verifique o texto inserido em um comentário. Se a frase contiver palavras inadequadas, ela não será liberada para publicação ou comentário. Caso contrário, a frase será liberada para postagem.

[input] -> frase inserida pelo usuário. Para realizar a verificação, o site realiza uma busca no banco de dados, comparando cada palavra da frase inserida com as palavras salvas no banco de dados que são consideradas ofensivas.

[output] -> frase que não contém nenhuma palavra inadequada é liberada para postagem.
